

## LG전자 고객언어 글쓰기 분석기 — 인터랙티브 웹 앱

Notion/Grammarly 스타일의 밝고 깨끗한 에디터 UI로, 텍스트를 입력하면 실시간으로 고객언어 4원칙 기반 오류를 탐지하고 개선안을 제시하는 앱.

### 페이지 구성

**단일 페이지 에디터 레이아웃 (2패널)**

- **좌측: 텍스트 에디터 영역**
  - 큰 텍스트 입력 영역 (textarea)
  - 오류가 있는 부분에 밑줄/하이라이트 표시 (빨강=오류, 주황=경고, 파랑=제안)
  - 상단에 전체 품질 점수 표시 (원형 게이지, 100점 만점)
  - 오류 유형별 카운트 배지 (구조/의미/문체/문법)

- **우측: 분석 결과 패널**
  - 각 오류를 카드 형태로 나열
  - 카드에는: 오류 유형 태그, 문제 설명, 원문 발췌, 개선 제안 2~3개
  - "수정 적용" 버튼 클릭 시 에디터에 자동 반영
  - 근거 원칙 표시 (예: "원칙1 - 고객 중심")

### 분석 엔진 (클라이언트 사이드 JavaScript)

제공해주신 9개 섹션의 규칙을 모두 JS 룰 엔진으로 구현:

1. **고객 중심 검사** — 기업 관점 표현("지원합니다", "제공됩니다", "탑재") 탐지, 기업명 첫 등장 경고
2. **쉬운 표현 검사** — 전문용어/한자어 사전 기반 탐지, 대체어 제안 (기동→켜져요, 탑재→들어 있어요 등)
3. **간결성 검사** — 명사 3개 이상 연속 탐지, 불필요 조사(적/것/들) 탐지
4. **정확성 검사** — 맞춤법 오류 사전(몇일→며칠, 안되요→안돼요 등), 띄어쓰기 규칙
5. **금지 표현(블랙리스트)** — "업계 최초", "완벽하게", "압도적" 등 탐지
6. **추천 표현(화이트리스트)** — 개선안 제시 시 고객 경험 중심 문장 패턴 우선 추천

### 점수 산정
- 100점에서 오류 심각도별 감점 (오류 -5, 경고 -3, 제안 -1)
- 원칙별 준수율을 4개 막대 차트로 시각화

### 디자인
- 흰색 배경, 부드러운 그림자, 라운드 카드
- 에디터는 큰 폰트(16px+), 넉넉한 line-height
- 오류 카드에 색상 코드된 좌측 보더 (빨강/주황/파랑)
- 상단 헤더에 "LG 고객언어 글쓰기 분석기" 타이틀
